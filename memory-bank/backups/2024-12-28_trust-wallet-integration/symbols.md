## 🎭 Framework Commands (框架命令)

- `/start` = 启动框架初始化
- `/r` 或 `/research` = 切换到研究模式
- `/i` 或 `/innovate` = 切换到创新模式
- `/p` 或 `/plan` = 切换到规划模式
- `/e` 或 `/execute` = 切换到执行模式
- `/rev` 或 `/review` = 切换到审查模式

- ↪ = 触发/导致 (Triggers)
